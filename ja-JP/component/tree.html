ツリー #

データの集合を階層化して表示します。

基本的な使い方 #

基本的なツリー構造。

遅延モードでのカスタムリーフノード #

チェックボックス無効化 #

ノードのチェックボックスを無効に設定することができます。

デフォルト展開、チェック #

ツリーノードは、最初に展開またはチェックすることができます。

ツリーノードのチェック #

カスタムノードの内容 #

ツリーノードの内容をカスタマイズできるので、好きなようにアイコンやボタンを追加することができます。

ツリーノードフィルタリング #

ツリーノードはフィルタリング出来る。

アコーディオン #

一度に展開できるのは同一レベル内の1つのノードのみです。

ドラッグ可能 #

ツリーノードに draggable 属性を追加することで、ツリーノードをドラッグ＆ドロップすることができます。
