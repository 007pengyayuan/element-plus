快速开始 #

本节介绍如何在您的项目中使用 Element Plus。

用法 #

完整导入 #

如果你对打包后的文件大小不是很在乎，那么使用完整导入会更方便。

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')

按需导入 #

您需要使用额外的插件来自动导入用了的组件。

自动导入 推荐 #

首先你需要安装 unplugin-vue-components。

npm install unplugin-vue-components

然后将下面的代码添加到 Vite 或 webpack 的配置文件。

Vite #
// vite.config.ts
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default {
  plugins: [
    // ...
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}
Webpack #
// webpack.config.js
const components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports =
  // ...
  plugins: [
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}

想了解其他的打包工具(Rollup, Vue CLI)，请参考 unplugin-vue-components。

手动导入 #

Element Plus 默认支持基于 ES Module 的 Tree Shaking。

但是您需要安装 unplugin-element-plus 来进行样式导入。 如何配置请查看文档。

App.vue

<template>
  <el-button>I am ElButton</el-button>
</template>
<script>
  import { ElButton } from 'element-plus'
  export default {
    components: { ElButton },
  }
</script>

// vite.config.ts
import ElementPlus from 'unplugin-element-plus/vite'

export default {
  plugins: [ElementPlus()],
}

快捷搭建项目模板 #

Vue CLI #

我们为 Vue CLI 准备了相应的 Element Plus Vue CLI 插件。 对于 Vue CLI，你可以用它们快速地搭建一个基于 Element Plus 的项目。

使用 Starter Kit #

我们提供了通用的项目模板，你可以直接使用，另外我们还提供了 Vite 模板。 对于 Laravel 用户，我们也准备了相应的模板，同样可以直接下载使用。

全局配置 #

当注册元素加速度时， 您可以通过一个带有 大小 和 的全局配置对象zIndex 来设置默认值 大小 表单组件. 和 zIndex 用于 弹出组件，默认值为 zIndex 为 2000。

完整引入 ElementPlus：

从 'vue' 导入 { createApp }
从 'element-plus'
从 './App.vue' 导入应用程序

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })

即时：

从 'vue' 导入 { createApp }
从 'element-plus' 导入 { ElButton }
从 './App 导入应用。

const app = createApp(App)
app.config.globalProperties。$ELEMENT = Power
  // options
}
app.use(Elbutton)

使用 Nuxt.js #

按照以上设置，项目中所有拥有 size 属性的组件的默认尺寸均为 'small'，弹框的初始 z-index 为 3000。
